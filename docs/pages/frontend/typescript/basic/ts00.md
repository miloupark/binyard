타입스크립트는 자바스크립트를 더 안전하게 사용할 수 있도록 "타입 관련 기능들을 추가한" 언어이다.

```js
// JS
let a = 1;
let b = 2;

console.log(a + b);
```

```ts
// TS
let a: number = 1;
let b: number = 2;

console.log(a + b);
```
