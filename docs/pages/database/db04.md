## 테이블 만들기 (1NF)

![](./images/t01.png)

#### 정규화 Normalization

정규화란 데이터베이스의 데이터를 체계적으로 정리하는 과정이다.

#### 정규 형식 Normal form

정규적인, 즉 규정에 맞고 정상인 형식

### 정규화 장점

- 데이터 중복 제거
- 데이터 무결성 준수
- 이상 현상 방지
